```javascript
var vm=new Vue({
                data(){
                    return{
                        msg:"字符串",
                        arr:["雷浪","友谊","追梦","123"],
                        num:1,
                        num2:2,
                        user:{
                            name:"肖雅涵",
                            age:"18"
                        }
                    }
                }
                
            }).$mount("#app");//挂载元素
            
            
//可以写表达式
<h2>{{num>num2 ? "是" : "不大于"}}</h2>//可以写三元表达式
<h3>{{user.name}}</h3>//点操作取值
<h3>{{arr.length}}</h3>//数组下标取值
<h2>{{arr[1]}}</h2>//数组的索引取值
//可以输出数组长度 arr.length
<h2>{{msg + num2}}</h2>//可以拼接
<h4>{{msg.split('').reverse().join(' ')}}</h4>//支持数组字符串的方法（join split......）
//每一个大括号里面只能包含单个表达式

<p>{{null}}</p>//（null 和undinfinde 都不会输出 ）
```

两个实例是接管不同的dom元素，所以修改one的数据时，two实例不受影响

如何在one实例修改two实例的数据？

定义一个全局变量json对象，添加name属性并赋值给实例one和two

```
        var obj = {
            name: 'obj'
        }
        let app = new Vue({
            name: 'vm', //默认root 修改名字
            data: obj
        }).$mount('#app');//挂载元素

        let app1 = new Vue({
            name: 'app1', //默认root 修改名字
            data: obj
        }).$mount('#app1');//挂载元素
```
